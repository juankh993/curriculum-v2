# Juan Carlos Hincapié — Portafolio Personal

Sitio web de portafolio y hoja de vida de Juan Carlos Hincapié Garzón, Desarrollador Front-End con +2 años de experiencia en agencias digitales.

## Stack

- **HTML5** semántico
- **CSS3** — Custom Properties, Grid, Flexbox, animaciones CSS puras
- **JavaScript Vanilla** — DOM, event listeners
- **AOS** (Animate On Scroll) v2.3.4
- **Inter** — tipografía vía Google Fonts

Sin frameworks, sin bundlers, sin dependencias de build.

## Estructura

```
cv/
```

## Diseño

Tema Apple/Google light:
- **Fondo:** `#ffffff` / `#f5f5f7`
- **Texto:** `#1d1d1f`
- **Acento:** `#0071e3` (Apple blue)
- **Indigo:** `#5e5ce6`
- Header: frosted glass blanco (`rgba(255,255,255,0.85)` + `backdrop-filter: blur`)
- Banner: dark navy `#080c18` con grid CSS + glow orbs

Los iconos de contacto usan SVG inline con colores de marca oficiales (Gmail `#EA4335`, LinkedIn `#0077B5`, WhatsApp `#25D366`).
